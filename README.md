# AI SEO Research

This repository contains research on AI-powered SEO content production for B2B SaaS. It compiles insights from leading SEO practitioners through their recent LinkedIn posts and YouTube discussions.

## Research Objective

The goal of this project is to understand how artificial intelligence is transforming SEO, Answer Engine Optimization (AEO), Generative Engine Optimization (GEO), AI Search, content production, and content distribution for B2B SaaS companies.

## Repository Structure

research/
├── sources.md
├── expert-profiles/
├── linkedin-posts/
└── youtube-transcripts/

- **sources.md** — List of the selected experts and why they were chosen.
- **expert-profiles/** — Background information and reference profiles for each expert.
- **linkedin-posts/** — Summaries and key takeaways from two recent LinkedIn posts for each expert.
- **youtube-transcripts/** — Summaries and key insights from one relevant YouTube video or interview for each expert.
